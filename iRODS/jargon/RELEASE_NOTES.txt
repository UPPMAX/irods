Jargon 2.2.0 Release Notes
================================================================================================

Project: Jargon - IRODS Java API
Date: 11/12/2009
Release Version: 2.2.0

This release contains the most current Jargon content from the main IRODS Subversion repository:
	svn://irodssvn.ucsd.edu/trunk/iRODS/jargon
and contains and subsumes all content from the previous repository on Google Code:
	http://code.google.com/p/extrods/
The Google Code repository is deprecated with this 2.2.0 release of Jargon. 

This release aligns the Jargon version number with the current IRODS version number, and contains bug fixes and patches applied
to Jargon since the release of 2.0.5 on Apr 08, 2009.  

================================================================================================
Requirements

-Jargon depends on Java 1.5+
-Jargon is built using Apache ANT

libraries
---------

-gsi-classes.jar (packaged in lib directory)

================================================================================================
Features and Changes 

This release is meant to clarify Jargon versioning since the 2.0.5 release.  Included in this release is a file,
jargon.extrods.googlecode.log, that documents changes in the Google repository.  The following changes occurred after the
release of Jargon 2.0.5 on April 8, 2009:
